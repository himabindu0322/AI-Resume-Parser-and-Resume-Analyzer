# AI-Resume-Parser-and-Resume-Analyzer
AI-powered Resume Parser and Analyzer built using NLP, Machine Learning, and Streamlit. Upload PDF resumes to get resume category prediction, skill extraction, resume scoring, and AI-based recommendations using TF-IDF and SVM models.

This application allows users to upload PDF resumes and get intelligent resume analysis including:

Resume Category Prediction,
Skill Extraction,
Resume Scoring,
AI-Based Recommendations

The project uses TF-IDF vectorization and SVM classification for resume categorization.

# How It Works

User uploads a PDF resume

PDF text is extracted using pdfplumber

Resume text is cleaned using NLP preprocessing

TF-IDF converts text into vectors

SVM model predicts resume category

Skills are extracted from resume

Resume score is calculated

AI recommendations are displayed
